# Трансферы в аэропорты Москвы

Одностраничный сайт для бронирования трансферов в аэропорты Москвы (Шереметьево, Домодедово, Внуково).

## Что на странице

- **Онлайн-бронирование** — кнопка открывает виджет бронирования
  (`https://transfer.yarobltour.ru/bron/bron_modern.php`) в модальном окне.
- **Виды трансфера** — индивидуальный и групповой «от дома», с тарифами
  (данные с [proline24.ru](https://proline24.ru/)).
- **О компании** — краткое описание и ссылка на [proline24.ru](https://proline24.ru/).

## Структура

Статический сайт без сборки и зависимостей:

- `index.html` — главная страница
- `styles.css` — все стили (общие для главной и посадочных страниц)
- `script.js` — логика модального окна бронирования
- `sitemap.xml` / `robots.txt` — для поисковых систем

### Посадочные страницы по городам

Каждый маршрут «город → Москва» — отдельная точка входа из поиска
(`<slug>/index.html`) с уникальным текстом (500+ слов: расстояние, время в
пути, точки подачи, аэропорты), уникальными title/H1/description, разметкой
LocalBusiness + Service + BreadcrumbList + FAQPage и внутренней перелинковкой.
Ссылки собраны в блоке «Направления» на главной. Страницы по конкретным
аэропортам — второй уровень, планируются позже.

- `transfer-yaroslavl-moskva/` — Ярославль → Москва (с тарифами)
- `transfer-rybinsk-moskva/` — Рыбинск → Москва
- `transfer-kostroma-moskva/` — Кострома → Москва
- `transfer-vologda-moskva/` — Вологда → Москва
- `transfer-tutaev-moskva/` — Тутаев → Москва
- `transfer-pereslavl-moskva/` — Переславль-Залесский → Москва
- `transfer-rostov-moskva/` — Ростов Великий → Москва

Главная (`index.html`) остаётся общим хабом «трансферы в аэропорты Москвы».
Цены на страницах городов, кроме Ярославля, — «рассчитывается при
бронировании» (реальных тарифов по этим маршрутам пока нет).

## Локальный запуск

```bash
python -m http.server 8099
# затем откройте http://localhost:8099
```
